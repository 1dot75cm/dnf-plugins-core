0.1.1 Release Notes¶

Provides DNF protected_packages Plugin and a bugfix to the Copr plugin.

Bugs fixed in 0.1.1:

  • Bug 1049310 - Keep running kernel
  • Bug 1104088 - Failed loading plugin: copr / ImportError: No module named requests
  • Bug 1111855 - DNF needs to protect the basesystem

0.1.0 Release Notes¶

This release provides the DNF repoquery Plugin and a bugfix for the DNF builddep Plugin.

Bugs fixed for 0.1.0:

  • Bug 1045078 - [rfe] repoquery
  • Bug 1103906 - dnf builddep command not working
