0.1.4 Release Notes¶

Provides DNF needs-restarting Plugin and DNF reposync Plugin.

Bugs fixed in 0.1.4:

  • Bug 1139738 - [rfe] reposync
  • Bug 1144003 - [abrt] dnf: download.py:181:_get_query_source:TypeError: ‘NoneType’ object has no attribute ‘__getitem__’

0.1.3 Release Notes¶

Added info switch to repoquery

Bugs fixed in 0.1.3:

  • Bug 1135984 - rfe: add repoquery -i support
  • Bug 1134378 - [abrt] dnf: copr.py:147:run:KeyError: ‘repos’
  • Bug 1123886 - [abrt] dnf: i18n.py:138:_exact_width_char:TypeError: must be unicode, not str

0.1.2 Release Notes¶

Bugs fixed in 0.1.2:

  • Bug 1108321 - debuginfo-install can’t install anything if debuginfo with version as installed not found
  • Bug 1116389 - dnf copr enable - 404 for repo without builds
  • Bug 1118809 - [abrt] dnf: generate_completion_cache.py:55:sack:UnicodeDecodeError: ‘ascii’ codec can’t decode byte 0xc3 in position 22: ordinal not in range(128)

0.1.1 Release Notes¶

Provides DNF protected_packages Plugin and a bugfix to the Copr plugin.

Bugs fixed in 0.1.1:

  • Bug 1049310 - Keep running kernel
  • Bug 1104088 - Failed loading plugin: copr / ImportError: No module named requests
  • Bug 1111855 - DNF needs to protect the basesystem

0.1.0 Release Notes¶

This release provides the DNF repoquery Plugin and a bugfix for the DNF builddep Plugin.

Bugs fixed for 0.1.0:

  • Bug 1045078 - [rfe] repoquery
  • Bug 1103906 - dnf builddep command not working
