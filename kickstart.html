DNF kickstart Plugin¶

Install packages according to Anaconda kickstart file.

Synopsis¶

dnf kickstart <ks-file>

Arguments¶

<ks-file>
Path to the kickstart file.

Examples¶

dnf kickstart mykickstart.ks
Install the packages defined in mykickstart.ks.
